# AURA - AI Unified Response Assistant 🤖🎙️

AURA is a Python-based intelligent voice assistant designed to provide hands-free computer interaction through speech recognition and automation.

The assistant listens to user commands, converts speech into text, processes the request, and responds using natural voice output. It can perform tasks such as web searching, retrieving information, opening applications, and providing real-time responses.

## Features

- 🎤 Voice command recognition
- 🔊 Text-to-speech responses
- 🌐 Web search automation
- 📚 Wikipedia information retrieval
- ⏰ Time and date assistance
- 💻 Application launching
- ⚡ Hands-free computer interaction

## Technologies Used

- Python
- SpeechRecognition
- SoundDevice
- Pyttsx3
- Wikipedia API
- Web Automation
